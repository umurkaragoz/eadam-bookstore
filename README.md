# E-Adam Bookstore


## Description

E-Adam Bookstore is a sample project for E-Adam Software Consultancy Agency


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Navigate to `localhost:3000/api` to use interact with the application using built-in Swagger interface.

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
